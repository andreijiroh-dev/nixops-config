# Nix flakes-packaged utilities

## What's included

To install them, run `nix profile add github:andreijiroh-dev/nixops-config#<package-name>` or
add them into your NixOS/home-manager package lists after importing the flake.

* [`coolify-compose`](./coolify-compose.nix)
* [`detect-vscode-for-git`](./detect-vscode-for-git.nix)
* [`ssh-agent-loader`](./ssh-agent-loader.nix)
